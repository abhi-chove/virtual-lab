Q1. Column chromatography is ____________ type of chromatography.
A    Adsorption
B    Partition
C    Gel
D    Gas
Ans is A

Q2. Chromatography technique was discovered by ____________
A    Pasture
B    Faraday
C    Martin
D    tswett
Ans is D

Q3. The solution used for elution is known as _________
A    mixture
B    elvent
C    eluate

Ans is B

Q4. In partition column chromatography which of the following stationary phase is preferred?
A    cellulose
B    alumina
C    magnesium silicate
D    charcol
Ans is A

A    charcol
B    ion exchange res in
